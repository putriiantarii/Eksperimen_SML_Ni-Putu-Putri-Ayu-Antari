# Eksperimen_SML_Ni-Putu-Putri-Ayu-Antari

Repository ini berisi eksperimen dataset, preprocessing, dan model machine learning untuk submission SMSML.

## Struktur Folder

- `preprocessing/` : Notebook eksperimen & file otomatisasi preprocessing
- `heart_preprocessing/` : Dataset hasil preprocessing
- `Membangun_model/` : Script modelling.py, modelling_tuning.py, artefak MLflow
- `.github/workflows/` : Workflow GitHub Actions
